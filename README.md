# 👥 Persona Users List Frontend

A modern Users List Frontend application built using **Next.js** that connects with the **Persona Backend API** developed using **React.js, Node.js, and Express.js**.  

This project provides an interactive interface to manage and display user data efficiently with a clean and responsive UI.

---

## 🚀 Features

- 👤 Display Users List
- 🔍 Search & Filter Users
- ➕ Add New Users
- ✏️ Edit User Details
- ❌ Delete Users
- 🔄 API Integration with Persona Backend
- 📱 Fully Responsive Design
- ⚡ Fast Rendering with Next.js

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/persona-users-frontend.git
```

### 2️⃣ Navigate to Project Directory

```bash
cd persona-users-frontend
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Configure Environment Variables

Create a `.env.local` file in the root directory and add:

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

---

## ▶️ Run the Application

### Start Development Server

```bash
npm run dev
```

The application will run at:

```bash
http://localhost:3000
```

---

## 🔗 Backend Connection

This frontend application connects with the **Persona Backend API** built using **Express.js**.

Make sure the backend server is running before starting the frontend.

Example Backend URL:

```bash
http://localhost:5000
```
